# 🎓 Student Performance Predictor

## 📌 Overview

The **Student Performance Predictor** is a machine learning project that estimates a student's final score based on daily habits such as study time, attendance, sleep, and previous academic performance.

This project is based on a simple idea:

*“Do everyday habits really impact how well a student performs?”*

Using machine learning, this project attempts to answer that question by predicting scores from user inputs.
